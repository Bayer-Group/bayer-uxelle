# UXElle (Early Preview)

UXElle is a React component library and design system for Bayer.

## Structure

This is a monorepo managed with npm workspaces:

- `packages/components` - `@uxelle/components` - Core React components
- `apps/storybook` - Storybook for component development and documentation

## Getting Started

```bash
# Install dependencies
npm install

# Start Storybook for development
npm run dev

# Build all packages
npm run build
```
